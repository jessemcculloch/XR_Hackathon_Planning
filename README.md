- Communications

    - Social Media

- Applications
- Workshops
- Equipment
- Sponsorships
- Scholarships
- Food and Drink
- Judging
- Prizes
- Opening Ceremony
- Closing Ceremony
- Mentors
- Shipping
- Showcase/Expo
- Schedule
- Helpful Tools
    - DevPost
